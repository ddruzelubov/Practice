# Practice

## Последовательность команд Git

Этот файл содержит последовательность команд Git.

### Клонирование репозитория

```bash
git clone https://github.com/ddruzelubov/Practice.git
cd Practice
```
### Создание веток
```bash
git branch homework1
git branch homework2
git branch homework3
git branch homework4
git branch node-hw-1
git branch node-hw-2
git branch node-hw-3
git branch node-hw-4
```

### Домашнее задание 1 (homework1)

```bash
git checkout homework1
git add homework.sql
git commit -m "Added answer for first homework"
git push origin homework1
```
### Домашнее задание 2 (homework2)

```bash
git checkout homework2
git add homework2/
git commit -m "Added answer for second homework"
```
